About me

I'm a game developer and experimenter who learns best when I'm actually building something — not reading about it.

I'm doing B.Tech in CSE with a Gaming Technology specialisation at VIT Bhopal. Game dev clicked for me because it forces you to care about physics, AI, rendering, and user feel all at once. That intersection is what keeps me genuinely interested.

I know C++, C#, Python, SQL, HTML, CSS, JavaScript, Git&Github, Unity Game Engine, Unreal Engine 5. For web stuff I tend to learn sideways — picking up whatever the project needs.
